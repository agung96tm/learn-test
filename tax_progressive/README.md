Tax Progressive
=================================================


Question:
```
1 - 30.000.000 = 5%
30.000.001 - 100.000.000 = 10%
100.000.001 and so on = 20%
```

Answer:
```
input: 24000000 
output: 24.000.000 * 5% = 0

input: 50000000
output: 30.000.000 * 5% + 20.000.000 * 10% = 0


input: 140000000
output: 30.000.000 * 5% + 70.000.000 * 10% + 40.000.000 * 20% = 0
```
