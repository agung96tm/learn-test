Pyramid Rotation
===========================

#### Question
```
Pyramid with Rotation, input height and Rotation
Example:

1. With input 3 and 0, then return strings:
  *
 ***
******

2. With input 3 and 1, then return strings:
*
**
***
**
*

3. With input 3 and 2, then return strings:
*****
 ***
  *

4. With input 3 and 3, then return strings
  *
 **
***
 **
  *

5. With input 3 and 4, then returns strings (support 4 = 0, same result)
  *
 ***
*****

6. With input 3 and -1, then returns strings (support reversed by negative number)
  *
 **
***
 **
  *
```


#### Answer
```
height: 3
rotation: 0
  *
 ***
******

height: 3
rotation: -1
  *
 **
***
 **
  *
  
height: 3
rotation: 1
*
**
***
**
*
```
