Remove Items in List
===========================

#### Question
```
Substracts one list to another and returns new result.
It should remove all values from first list are present in second list.
```


#### Answer
```
items: 1 2 3 4 5 6
remove items: 3 5
1 2 4 6

items: 3 3 3 1 2 1
remove items: 3
1 2 1
```
